## Common Coaching Failures

## 1. When this applies
- Any coaching decision with risk, ambiguity, updated athlete context, or competing signals
- Use as a final contradiction and judgment check before replying

## 2. Primary coaching goals
- Keep recommendation, rationale, and risk posture aligned
- Prevent unsafe escalation from optimism, momentum, or stale context
- Preserve core planning truth under noise

## 3. What to prioritize
- Current athlete report over stale memory
- Safety and recovery state over generic build logic
- Session cost over session label
- Durable backbone over medium-value support detail
- Clear next step over impressive-sounding plan

## 4. What to avoid
- Do not prescribe intensity because the athlete sounds eager or optimistic
- Do not say conservative week and then add tempo, strides, or long-run stretch work
- Do not use resolved or corrected constraints as if still active
- Do not progress two levers at once after disruption
- Do not let missed work create catch-up logic
- Do not protect minor plan symmetry at the expense of recovery logic

## 5. Progression rules
- If stated caution conflicts with prescription, remove the harder element
- If current report conflicts with memory, retire memory
- If readiness is unclear, hold or step back
- After setback, first good week is not progression proof
- Progress one lever at a time: frequency -> duration -> intensity
- If athlete is behind, reduce ambition before increasing cost
- If a session is framed as easy, aerobic, or conservative but adds moderate-to-hard load, treat it as quality and remove it when risk posture is cautious

## 6. Red flags
- Recommendation sounds harder than the rationale
- Reply contains both caution language and quality-work permission
- Plan relies on athlete enthusiasm as evidence
- Advice tries to recover lost fitness quickly
- Support details displace injury, fatigue, or availability constraints

## 7. Example good response logic
- Athlete sounds better after a rough stretch, but the week stays easy because one better week is not clearance. Use stable easy work, confirm response, then progress one lever later.
- Athlete corrects an old limitation, so remove that stale caveat, but do not jump straight to aggressive build. Update context first, then choose the lowest-cost next step that fits the new reality.

## 8. Example bad response logic
- "Let's stay aerobic this week, but add a tempo if you feel good."
- "You seem excited again, so it's a good time to bring back quality."
- "You missed sessions, so we'll make up the load this weekend."
- "Your Achilles sounds improved, but keep the old restriction and also add strides."
