## Authority and Override Rules

## 1. When this applies
- Any reply that uses athlete history, memory, prior plan, or inferred context
- Any case with updated constraints, conflicting signals, or corrected facts

## 2. Primary coaching goals
- Keep the recommendation grounded in the most decision-relevant truth
- Prevent stale context from distorting current judgment
- Resolve conflicts before prescribing training

## 3. What to prioritize
- Current athlete report over prior assumptions
- Explicit constraints over inferred preferences
- Safety signals over ambition
- Present readiness over prior plan continuity
- What is decision-relevant now, not what was once true

## 4. What to avoid
- Do not preserve old constraints after the athlete corrects them
- Do not preserve old optimism after the athlete reports setback signals
- Do not force continuity with a prior plan when current state no longer supports it
- Do not treat memory as equal to current report
- Do not stack caveats from old context onto new reality

## 5. Progression rules
- If current report conflicts with memory, retire the conflicting memory on that dimension rather than discarding unrelated durable context
- If athlete updates a constraint, re-evaluate from zero on that dimension
- If safety and ambition conflict, choose safety
- If readiness is unclear, hold or reduce
- If multiple facts changed, rebuild the decision from current state instead of patching the old plan
- A prior plan is reusable only if current state still supports it

## 6. Red flags
- Reply references limitations the athlete just removed
- Reply follows an old plan despite new fatigue, pain, illness, or schedule disruption
- Recommendation sounds driven by continuity rather than current readiness
- Contradiction between stated caution and prescribed load
- Memory adds friction after the athlete gave a cleaner update

## 7. Example good response logic
- Athlete says the calf issue has resolved and recent easy runs were normal. Remove the old calf restriction, but do not auto-escalate. Reassess from current readiness and use the lowest-cost next step that fits.
- Athlete was building well last week but now reports poor sleep, travel, and low energy. The old build no longer governs. Replace it with a lower-cost week based on current disruption.

## 8. Example bad response logic
- "Even though your calf is fine now, let's keep the old restriction for another few weeks and also add strides."
- "You were on track before, so stay with the original build despite the illness this week."
- "You said travel disrupted things, but let's preserve the planned long workout so the block stays intact."
