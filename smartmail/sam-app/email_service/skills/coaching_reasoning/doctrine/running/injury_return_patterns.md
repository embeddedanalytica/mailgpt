## Running Injury Return Patterns

APPLIES WHEN: Runner returning from pain, niggle, flare, or recent reduced running; issue improving or recently quiet but tolerance not yet proven

## Rules
- Green = no worsening during run and no worse next morning
- Yellow = discomfort that settles quickly and does not carry over
- Red = pain changes mechanics, grows during run, or is worse next morning
- Stay in easy running until repeated green exposures
- Build frequency first, then duration, then intensity
- First intensity is light and isolated, not a full workout
- If any run turns red, step back immediately
- First good week is fragile — repeat before progressing
- Athlete should be pain-free during easy running before any quality sessions

## Avoid
- Using one good run as clearance
- Adding intensity because pain is better
- Combining longer run + faster work in the same return phase
- Treating strides as automatically safe
- Chasing pre-injury volume too quickly

## Style
- Rebuild run tolerance without re-aggravation
- Restore confidence before normal training ambition
- Separate symptom relief from true load readiness
- Focus on pain during run, warm-up behavior, and next-morning response

## Red Flags
- Pain in warm-up that does not fade
- Pain returns later the same day or next morning
- Runner wants to test fitness early
- Uneven terrain, hills, speed, or long run added too soon
- "It feels mostly gone" after recent inconsistency
