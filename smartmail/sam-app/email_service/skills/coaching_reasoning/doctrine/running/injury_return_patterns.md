## Running Injury Return Patterns

## 1. When this applies
- Runner is returning from pain, niggle, flare, or recent reduced running
- Issue is improving or recently quiet, but tolerance is not yet proven

## 2. Primary coaching goals
- Rebuild run tolerance without re-aggravation
- Restore confidence before normal training ambition
- Separate symptom relief from true load readiness

## 3. What to prioritize
- Pain during run, warm-up behavior, and next-morning response
- Frequency of safe exposures before bigger single runs
- Flat, controlled, low-cost running before complex stimuli
- Duration before intensity when symptoms are still echoing

## 4. What to avoid
- Do not use one good run as clearance
- Do not add intensity because pain is better
- Do not combine longer run + faster work in the same return phase
- Do not treat strides as automatically safe
- Do not chase pre-injury volume too quickly

## 5. Progression rules
- Green = no worsening during run and no worse next morning
- Yellow = discomfort that settles quickly and does not carry over
- Red = pain changes mechanics, grows during run, or is worse next morning
- Stay in easy running until repeated green exposures
- Build frequency first, then duration, then intensity
- First intensity is light and isolated, not a full workout
- If any run turns red, step back immediately
- First good week is fragile; repeat before progressing

## 6. Red flags
- Pain in warm-up that does not fade
- Pain returns later the same day or next morning
- Runner wants to test fitness early
- Uneven terrain, hills, speed, or long run added too soon
- Language like "it feels mostly gone" after recent inconsistency

## 7. Example good response logic
- The calf sounds improved, but tolerance is not proven yet. Keep several easy green runs first, then extend one run slightly. Add intensity only after stable easy running, and start with a small exposure, not a full quality session.

## 8. Example bad response logic
- "Since it felt good this week, bring back tempo."
- "Let's test it with a long run and see."
- "Pain is lower now, so strides are fine."
