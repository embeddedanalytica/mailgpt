## Common Running Prescription Errors

## 1. When this applies
- Any running recommendation involving load, workout choice, comeback, or progression
- Use as a final check before prescribing a run session

## 2. Primary coaching goals
- Prevent session choice errors that outrun current tolerance
- Keep running progression specific, conservative, and internally consistent
- Reduce avoidable flare-ups from common strategist mistakes

## 3. What to prioritize
- Current run tolerance over target race logic
- Session cost over session name
- Repeatable easy running over isolated ambitious sessions
- Mechanical simplicity: flat, controlled, low-variance running first
- The cheapest next useful step

## 4. What to avoid
- Do not prescribe workouts just because the athlete is behind
- Do not hide intensity inside "steady," "moderate," "progression," or "strides"
- Do not pair long run growth with workout return
- Do not treat one smooth run as proof
- Do not use hills, terrain, or pace goals early in a fragile phase
- Do not let race timeline override current tolerance

## 5. Progression rules
- If tolerance is uncertain, keep the run easy and predictable
- Add only one cost at a time: frequency or duration or intensity
- Before quality, require stable easy running and normal next-day response
- First quality exposure is small, controlled, and isolated
- If athlete absorbs a week well, repeat once before a bigger jump when risk is recent
- If response is mixed, remove complexity before removing volume

## 6. Red flags
- "Let's test it"
- "You handled one run well"
- "We'll make up fitness"
- Workout added after inconsistent week
- Long run stretched after recent pain, illness, or travel
- Pace prescription despite unstable energy or symptoms

## 7. Example good response logic
- Athlete wants structure back after an uneven stretch. Keep the week anchored by easy runs, add only a small amount of duration if recent response was clean, and delay workout return until easy running is stable enough that the next-day response is boring.

## 8. Example bad response logic
- "You've been inconsistent, so let's re-establish fitness with a long run and tempo."
- "The niggle seems better, so add strides at the end."
- "You only have a few weeks left, so we need to push now."
